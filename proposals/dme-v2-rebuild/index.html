05 · HubSpot Integration

What we handle — and what you'll need to provide.

HubSpot stays your CRM and system of record. The new site is downstream of HubSpot, never competing with it. Most of the integration work is on our side — but we'll need a small number of things from your team to wire it up cleanly.

/ Our side

What Socia Visual handles

Everything technical on the WordPress side, end to end:

  • → Drop the HubSpot tracking pixel site-wide
  • → Embed HubSpot forms (Apply, Contact, Camp Inquiry, Newsletter)
  • → Style forms to match the new DME design system
  • → Embed HubSpot meeting links (admissions consults)
  • → Wire HubSpot chat widget if you use one
  • → Test every form end-to-end into HubSpot CRM
  • → Confirm conversion events fire correctly in HubSpot reporting
/ Your side

What we'll need from you (≈2–3 hrs total)

Small but important — most of these are 5-minute lookups in HubSpot:

  • → HubSpot Portal ID / Hub ID (single number, lives in HubSpot Settings)
  • → Forms list — confirm or build the forms in HubSpot's form builder for: Apply, Contact, Camp Inquiry, Newsletter, plus any sport-specific inquiries
  • → Meeting links — which staff calendars are bookable for admissions consults? Provide the HubSpot meeting URLs
  • → Chat widget — yes/no, and if yes confirm which inbox routes
  • → CRM workflows — confirm the existing lead-routing workflows are pointed where you want them (your team owns the workflow logic; we just feed leads in)
  • → HubSpot CMS pages (if any) — decide whether any pages currently on HubSpot CMS get migrated to WordPress or stay on HubSpot

None of this is gating the build — we can start without it.
We'll just need this list before we deploy forms in week one.
